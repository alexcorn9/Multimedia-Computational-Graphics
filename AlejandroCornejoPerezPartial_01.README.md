# Image Editor - Partial_01

## Description
This program allows users to crop, invert, and rotate regions in a JPG image.

## How to Run:
Compile:
javac Parcial_01/*.java

Run:
java Parcial_01.ImageEditorApp

## Procedures:
- Crop selected region
- Invert colors in region
- Rotate region (90°, 180°, 270°)
- Save final image

## Program Structure:
The program is divided into two classes:
- ImageEditorApp: Handles menu and user interaction.
- ImageEditor: Handles image processing logic.

## Technical Concepts Used:
- BufferedImage manipulation
- Bit shifting
- Coordinate transformation
- Object-Oriented Programming
